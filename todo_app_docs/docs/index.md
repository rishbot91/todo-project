# Welcome to MkDocs

For full documentation visit [mkdocs.org](https://www.mkdocs.org).

## Commands

* `mkdocs new [dir-name]` - Create a new project.
* `mkdocs serve` - Start the live-reloading docs server.
* `mkdocs build` - Build the documentation site.
* `mkdocs -h` - Print help message and exit.

# Todo App Documentation

This project is a web-based To-Do List application backend, implemented using Django and Django REST Framework. It provides APIs for managing tasks with functionality to create, update, delete, and view tasks. The app supports user authentication and has a CI/CD pipeline for testing and deployment.
